# Comparison-of-Region-based-Sales-Tableau
This project focuses on analyzing and comparing sales performance between two selected regions using Tableau. The goal is to help stakeholders understand regional trends, identify performance gaps, and make better business decisions through visual analytics.

Project Overview

A dashboard was built to compare sales metrics for two user-selected regions. The dashboard allows interactive selection of regions and displays key performance indicators along with supporting visualizations such as maps, charts, and tables.
As described in the project document, the objective is to visualize sales comparison between two selected regions and provide insights that can support decision-making. 
Dataset
Sample Superstore dataset (Excel)
Contains information about orders, customers, products, and sales across different regions.
Features Implemented

The dashboard includes:
Primary and Secondary Region selection using parameters
First Order Date calculation
Total Sales
Average Sales per Order
Number of Customers
Number of Orders
Number of Products Sold
Map visualization for regional sales distribution
Tables and charts for category-wise and trend analysis
These elements follow the tasks outlined in the project instructions, including parameters, calculated fields, hierarchy creation, and dashboard design. 


Tools & Concepts Used :
Tableau Public
Parameters
Calculated Fields
Hierarchies
Dashboards
Maps and charts for visualization
Dashboard Insights

The dashboard enables:
Side-by-side comparison of two regions
Identification of high-performing product categories
Analysis of sales distribution across locations
Understanding order trends and customer activity
Project Outcome

The final dashboard provides a clear comparison of regional performance and demonstrates how visual analytics can be used to monitor sales and identify opportunities for improvement.
